# Markdown → 标准公文排版（Word/WPS）

将 Markdown 格式的公文草稿一键转换为符合 **《党政机关公文格式》**（GB/T 9704‑2012）的 **.docx** 文件，可直接在 Microsoft Word 或 WPS 中打开、打印。

## 快速开始

```bash
git clone https://github.com/ijerrywong/markdown-to-gongwen.git
cd markdown-to-gongwen
bash setup.sh
```

---

### 方式一：Typora 一键导出（推荐）

配置一次后，在 Typora 中写完公文草稿，直接 `文件 → 导出 → 公文导出` 即可生成排版好的 .docx。

#### 配置步骤

1. 打开 Typora → **偏好设置 → 导出 → 添加导出**
2. 类型选 **"自定义"**，在 **"命令"** 字段填入：

```
/你的完整路径/typora-gongwen.sh -d ~/Desktop
```

> 例如：
> ```
> /Users/jerry/Projects/markdown-to-gongwen/typora-gongwen.sh -d ~/Desktop
> ```

3. 保存后即可使用

#### 使用流程

1. 在 Typora 中按 Markdown 书写约定编写公文草稿
2. 按 **⌘+S** 保存文件
3. `文件 → 导出 → 公文导出`
4. 桌面自动生成与 .md 文件同名的 .docx（如 `通知.md` → `通知.docx`）

#### 参数说明

| 参数 | 作用 | 示例 |
|------|------|------|
| `-d <目录>` | 指定输出目录 | `-d ~/Desktop` 输出到桌面 |
| `-o <路径>` | 指定完整输出路径（优先级高于 -d） | `-o ~/Desktop/公文.docx` |
| 不加参数 | 输出到 .md 文件所在目录 | 直接运行 |

### 方式二：命令行终端

确保已创建虚拟环境并安装依赖（见快速开始），然后运行：

```bash
.venv/bin/python3 markdown-to-gongwen.py 输入.md [输出.docx]
```

示例：

```bash
.venv/bin/python3 markdown-to-gongwen.py example_gongwen.md 通知公文.docx
```

## Markdown 书写约定

| Markdown 标记 | 公文结构 | 示例 |
|--------------|---------|------|
| `# 标题` | 公文的**主标题**（居中，二号方正小标宋） | `# 关于加强管理的通知` |
| `## 标题` | 一级标题 "一、" | `## 工作目标` |
| `### 标题` | 二级标题 "（一）" | `### 重点任务` |
| `#### 标题` | 三级标题 "1." | `#### 细化措施` |
| `##### 标题` | 四级标题 "（1）" | `##### 责任分工` |
| 普通段落 | 正文（3 号仿宋，首行缩进 2 字符） | 直接书写 |
| `主送机关：……` | 主送机关（顶格，全角冒号） | `主送机关：各下属单位` |
| `成文日期：YYYY年M月D日` | 落款日期（右空四字） | `成文日期：2026年4月29日` |

> 主送机关和成文日期将自动从段落中剥离并格式化，不会重复留在正文中。

## 输出文档标准速览

- **纸张**：A4（210×297 mm）
- **版心**：156×225 mm
- **行数/字数**：每页 22 行，每行 28 字
- **正文字体**：3 号仿宋_GB2312（16 pt），行距固定 28 磅，首行缩进 2 字符
- **一级标题**：3 号黑体，缩进 2 字符
- **二级标题**：3 号楷体_GB2312，缩进 2 字符
- **三级/四级标题**：3 号仿宋_GB2312 加粗/常规，缩进 2 字符
- **主标题**：2 号方正小标宋简体（居中）
- **页码**：4 号半角宋体，左右一字线

## 常见问题

**Q：首次使用如何安装依赖？**
A：在项目目录运行 `bash setup.sh`，自动完成虚拟环境创建和依赖安装。

**Q：新电脑部署需要拷贝哪些文件？**
A：拷贝整个项目文件夹（不含 `.venv/` 和 `venv/`），然后在新电脑上运行 `bash setup.sh`。

**Q：没有"方正小标宋简体"字体会怎样？**
A：Word/WPS 会自动降级为类似宋体，建议在系统中安装该字体以获得最佳效果。

**Q：Typora 导出时报"未读取到 Markdown 内容"？**
A：请确保已在 Typora 中保存文件（⌘+S），再点击导出。脚本会自动检测 Typora 当前文档。

**Q：新电脑部署需要拷贝哪些文件？**
A：只需拷贝 `markdown-to-gongwen.py` 一个文件，确保已安装 Python 3.8+ 即可。
